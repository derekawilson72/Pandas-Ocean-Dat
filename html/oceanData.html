class oceanData.oceanDataPlot[source]¶

merge two datasets with intersecting (overlapping) date ranges. Plot the data as a single dataset and export as a netcdf.

plotting datasets of wind and wave data as well as the extreme events. Uses Pandas inherent time series plotting tools.

None. Uses class variables

out : fig A matplotlib figure object of the timeseries plot

>>> self.plotDates()

rosePlot(magn, dirs)[source]¶

generate a rose plot with the class values for WaveData or Winddata. A rose plot will display the magnitude and direction of both wind and waves. The rose plots will appear as wedges in the direction of the wave or wind. The magnitude of the wave or wind will appear as the diameter of the wedge and a cumulative distribution function will appear as color codes of the wedge.

magn: array-like magnitude of wind or waves dirs: array-like direction of wind or waves

out : fig A matplotlib figure object of the roseplot

>>> self.rosePlot(self.WaveData.Wvht, self.WaveData.Wvdr)
matplotlib fig

>>> self.rosePlot(self.WindData.Wnsp, self.WaveData.Wndr)
matplotlib fig

>>> self.rosePlot()  ##return random dataset for testing
matplotlib fig

Using the Code¶

See the oceanData.ipynb in ipython notebooks for a more illustrated example of the module. Roseplots were adapted from the Rose Plot module.

import oceanData.oceanDataPlot
odp1=oceanDataPlot()
odp1.intersectDates()
odp1.plotDates()
odp1.fig.show()

odp1.createWaveRose()
odp1.createWindRose()
